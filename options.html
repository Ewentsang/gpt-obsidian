Obsidian vaults

Connect a vault so this extension can save ChatGPT conversations into it. One-time setup:

  1. In Obsidian: install the Local REST API plugin, turn on Enable Non-encrypted (HTTP) Server, and copy its API key.
  2. Back here: click Detect running vaults, then paste the key and give the vault a name.
Using more than one vault?

Each vault runs its own server, so keep every vault you use at the same time on a different HTTP port (set it in that vault's Local REST API settings). The name is just a label for you — Obsidian's API can't report a vault's real name.
