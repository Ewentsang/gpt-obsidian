Obsidian vault connections

Each Obsidian vault runs its own Local REST API server on its own port with its own API key. Add one connection per vault. Vaults you keep open at the same time must each use a different port — set it in that vault's Local REST API plugin settings, with Enable Non-encrypted (HTTP) Server turned on. Give each connection a label so you can tell them apart (the API can't report the vault's name).
